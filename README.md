# Yii2 Shortlink Service

Test project: URL shortener with QR.

## Install
1. clone repository "git clone https://github.com/voronbeats/shortlink.git"
2. composer require endroid/qr-code
3. configure db in config/db.php
4. run migrate
5. php yii serve

## Features
- URL validation
- Availability check
- Short link + QR
- Redirect with logging
